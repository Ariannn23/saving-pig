# SAVING PIG - Frontend (apps/web)

Esta es la aplicación web de SAVING PIG, construida con React, TypeScript y Vite. Ofrece una interfaz premium y responsiva para la gestión de finanzas personales.

## Tecnologías Principales

- **React 18** + **TypeScript**
- **TailwindCSS**: Estilos premium con glassmorphism.
- **Zustand**: Gestión de estado global (Auth, UI).
- **React Query**: Sincronización y caché de datos.
- **Lucide React**: Iconografía moderna.
- **Recharts**: Visualización de datos y gráficos.

## Estructura de Carpetas

- `src/components`: Componentes UI reutilizables y específicos.
- `src/hooks`: Ganchos personalizados para lógica de negocio (hooks).
- `src/layouts`: Estructuras de página responsivas.
- `src/lib`: Configuraciones de clientes externos (Supabase).
- `src/pages`: Vistas principales de la aplicación.
- `src/services`: Capa de servicios para interactuar con la API.
- `src/store`: Estados globales compartidos.

## Cómo empezar

1. Asegúrate de tener configurado tu archivo `.env`.
2. Instala dependencias: `npm install`
3. Inicia en desarrollo: `npm run dev`
